# AGENT: i18n-agent

> 상위: `ORCHESTRATOR.md` · 참조: 06_FEATURES.md

## 역할
18개 언어 다국어 지원.

## 책임 범위
- UI 문자열 전부 키화(하드코딩 문구 금지).
- 어패럴/굿즈 명칭은 nameI18n 맵(03 문서)으로 관리.
- 로케일 자동 감지 + 유저 설정 저장(users.locale).
- 날짜/숫자/통화 로케일 포맷.

## 언어 목록 [가정 — 최종 확정 필요]
> 18개. 글로벌 주요 언어 기준 초안. 실제 타깃 시장에 맞춰 조정.
1. 한국어(ko) 2. 영어(en) 3. 일본어(ja) 4. 중국어 간체(zh-CN) 5. 중국어 번체(zh-TW)
6. 스페인어(es) 7. 포르투갈어(pt) 8. 프랑스어(fr) 9. 독일어(de) 10. 이탈리아어(it)
11. 러시아어(ru) 12. 인도네시아어(id) 13. 태국어(th) 14. 베트남어(vi) 15. 힌디어(hi)
16. 아랍어(ar, RTL) 17. 터키어(tr) 18. 네덜란드어(nl)

## 반드시 지킬 것 [확정]
- 아랍어 등 RTL 레이아웃 대응.
- 생성 프롬프트는 영어 기준 유지(모델 성능), UI만 번역.
- 번역 누락 시 영어 폴백.

## 기술 [가정]
- next-intl 또는 next-i18next.

## 확인 필요 [확실하지 않음]
- 18개 언어 최종 목록(타깃 시장 우선순위) → 사용자 확정 필요.
- 번역 품질(기계번역 vs 검수) 정책.
