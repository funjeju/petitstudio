# AGENT: admin-agent

> 상위: `ORCHESTRATOR.md` · 참조: 07_ADMIN_DASHBOARD.md, 05_COST_AND_TIERS.md · 스킬: cost-calculator

## 역할
어드민 대시보드 + 투자용 지표 로그.

## 책임 범위
- 회원 관리(목록/검색/등급변경/크레딧 조정/정지).
- 데이터 관리(어패럴·굿즈 CRUD, 피드 모더레이션).
- 생성 설정(기본 품질 티어, 등급별 생성량, kill switch, 환율).
- 지표 대시보드(성장/사용·원가/수익·전환/로그).

## 핵심 지표 [확정 구조]
- 상단 고정: **API 원가 실시간 누계**(폭주 조기경보 겸).
- MAU/DAU, 등급 분포, 리텐션.
- 총 생성 수(티어별), 유저당 원가, 무료 손실.
- 무료→Pro 전환율, MRR, 실효 마진.
- 굿즈 GMV/객단가/마진(인쇄단가 확정 후 [알 수 없음]).

## 반드시 지킬 것 [확정]
- role=="admin" 서버 재검증.
- 지표 집계는 캐시 문서(metrics/{period}) 권장(읽기비용 절감).

## 확인 필요 [확실하지 않음]
- 결제 로그 소스(Firestore vs PG 웹훅) → PG 선정 후.
- 투자자용 export(CSV/PDF) 후순위.
