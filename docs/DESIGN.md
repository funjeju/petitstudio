# DESIGN — 시안 3번 (PAWFOLIO / 모던 & 미니멀)

> 상위: `ORCHESTRATOR.md` · 담당: frontend-agent
> 첨부 4개 시안 중 **3번 "모던 & 미니멀 (화이트 + 그레이 톤)"** 채택.

---

## 1. 디자인 방향 [확정]

- 컨셉: "가장 깨끗한 한 장, 모든 스타일의 시작." 심플하고 세련된 화이트+그레이.
- 원칙: **UI는 물러나고 사진(결과물)이 주인공.** 채도 낮은 뉴트럴 배경 + 무채색 UI.
  - 근거: 반려동물 사진은 털색이 제각각이라 UI가 색을 쓰면 사진과 충돌. 뉴트럴이 어떤 아이든 주인공화.
- 톤: 프리미엄·미니멀. 가격 프레임을 올려 굿즈 결제까지 정당화(2번 핑크 대비 우위 — 제 판단).

## 3. 타이포그래피 [확정]
- 헤드라인: 산세리프 볼드(예: Pretendard/Inter). 시안 3의 "가장 깨끗한 한 장" 스타일.
- 본문: 산세리프 레귤러.
- 세리프는 사용하지 않음(시안 3는 모던 산세리프 기조).
- 문장 케이스, 과도한 볼드 지양.

## 4. 레이아웃 [확정]
- 코너: 카드 12px, 컨트롤 8px.
- 보더: 0.5px 헤어라인.
- 여백: 넉넉한 화이트스페이스(미니멀).
- 그림자: 최소(포커스 링 정도). 그라디언트/노이즈 금지.

## 5. 반응형 [확정]
- 모바일 우선. 브레이크포인트(가정): sm 640 / md 768 / lg 1024 / xl 1280.
- 모바일: 하단 5탭바. 웹: 좌측 사이드바 + 넓은 피드 그리드(2열→3~4열).
- 피드 카드 컴포넌트는 동일, 그리드 열 수만 반응형 변경.

## 6. 핵심 컴포넌트

### 피드 카드
- 이미지(카드의 약 90%) + 펫이름·품종 + 아이템 칩 + "내 아이에게 입혀보기" 버튼 + 좋아요/저장.
- "입혀보기" 버튼이 좋아요/저장보다 시각 우선순위 높음(전환 유도).

### 하단 탭바(모바일)
- 홈 / 탐색 / [중앙 원형 카메라 FAB] / 컬렉션 / 마이 스튜디오.
- FAB는 반쯤 떠 있는 원형, 아이콘은 카메라/스파클(생성 암시).

### 피팅룸
- 중앙 코어 이미지 + 하단 어패럴 레일 + 배경 선택 + 텍스트 입력.

## 7. 마이크로카피 톤 [가정]
- 유저 언어 사용: "코어 이미지 만들기" 대신 "우리 아이 촬영하기", "가상 피팅룸" 대신 "입혀보기".
- 펫 이름을 UI에 지속 노출("초코의 스튜디오", "초코의 1월 룩").

## 8. 제거/금지 (원 시안에서)
- 홈의 "코어/피팅룸/굿즈/주문" 4버튼 퀵메뉴 제거(06 문서).
- 홈 상단 가격표 달린 굿즈 진열 제거(컬렉션에서 파생).
- 과도한 프로모션 배너 금지.

## 9. 확인 필요 [확실하지 않음]
- 무채색 강조만으로 CTA가 약해 보이면 포인트 1색 도입 여부 → A/B.
- 폰트 라이선스(상용) 확정 필요.
