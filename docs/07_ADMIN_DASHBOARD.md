# 07. 어드민 대시보드 · 투자용 지표

> 상위: `ORCHESTRATOR.md` · 담당: admin-agent · 스킬: cost-calculator

---

## 1. 접근 제어 [확정]
- `users.role == "admin"` 인 계정만 접근.
- 어드민 API(`/api/admin/*`)는 서버에서 role 재검증(클라이언트 신뢰 금지).

## 2. 어드민 기능

### 2.1 회원 관리
- 유저 목록/검색(이메일·등급·가입일).
- 등급 수동 변경, 크레딧 수동 조정(고객대응).
- 유저 상태(정상/정지), 소프트 삭제 콘텐츠 관리.

### 2.2 데이터 관리
- 어패럴 카탈로그 CRUD(`apparel` 컬렉션, 다국어 명칭·anchor·프롬프트 조각).
- 굿즈 상품 CRUD(`goods`, 판매가·필요 이미지 수·생성원가).
- 피드 모더레이션(공개 착장 신고 처리).

### 2.3 생성 설정
- **기본 품질 티어 선택(low/medium/high)** — 결과물 보고 조정. `adminConfig.defaultQuality`.
- 등급별 월 생성량(`tierLimits`) 조정.
- 월 총액 kill switch 임계값(`monthlyCostCapUsd`).
- 환율(`usdKrw`) 갱신.

## 3. 투자용 지표 (대시보드 상단)

> 원가 실시간 누계는 최상단 고정(비용 폭주 조기경보 겸용).

### 3.1 성장 지표
- 누적 가입자 / MAU / DAU.
- 등급별 분포(무료/Pro/VIP 수·비중).
- 신규 가입 추이, 리텐션(W1/W2/W4).

### 3.2 사용/원가 지표 (핵심)
- 총 생성 이미지 수(품질 티어별).
- **실제 API 원가 누계**(costLogs 합산, 이번 달/전체).
- 유저당 생성 수 / 유저당 원가.
- 무료 티어 총 손실(무료 생성 × 실단가).
- edit 실단가 추적(파일럿→실운영 반영). [확실하지 않음]

### 3.3 수익/전환 지표
- 무료→Pro 전환율.
- 구독 매출(MRR), Pro/VIP 실효 마진.
- 굿즈 주문 건수 / GMV / 객단가.
- 굿즈당 마진(인쇄단가 확정 후). [알 수 없음]
- 이미지 1장당 매출 대비 원가(유닛 이코노믹스).

### 3.4 로그/감사
- 생성 로그(누가/언제/무슨 품질/얼마).
- kill switch 발동 이력.
- 비정상 사용(rate limit 초과) 이력.

## 4. 데이터 소스 매핑

| 지표 | 소스 컬렉션 |
|---|---|
| 가입/등급 분포 | `users` |
| 생성 수/원가 | `costLogs`, `generations` |
| 매출/전환 | `users.tier` 변경 이력 + 결제 로그 |
| 굿즈 | `orders`, `goods` |
| 설정 | `adminConfig` |

## 5. 구현 노트 [가정]
- 지표 집계는 실시간 쿼리 대신 일/시간 단위 집계 문서(`metrics/{period}`) 캐시 권장(읽기 비용 절감).
- 투자자 공유용 스냅샷 export(CSV/PDF) 기능 후순위.

## 6. 미확정 [확실하지 않음]
- 결제 로그를 Firestore에 둘지 결제사(PG) 웹훅으로 받을지 — PG 선정 후.
- 굿즈 마진 지표는 인쇄 제휴 단가 확정 전 "미정" 표기.
